This program uses the GuitarParty API to display how to play a guitar chord.

Tested On:
Node v8.12.0
Windows 10 Pro

To install npm modules execute:
>npm install

This will install the modules listed as dependencies in the package.json file.

To run either execute
>npm start
or 
>node server.js

Testing:
Use Chrome browser to visit:
Visit any of the following: 

http://localhost:3000

ISSUES:

The owner has discontinued the support of the API therefore some chords do not display and returns a 403 error.
